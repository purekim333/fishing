# 프로젝트 기능 요약

## 목적
애플 신제품 소개 페이지처럼 스크롤에 따라 섹션이 전환되는 구조로,
축제 당일에 필요한 핵심 정보만 빠르게 확인하도록 구성한 안내 페이지입니다.

## 핵심 기능

### 1) 실시간 상태 스트립
- 날씨, 도로 소요시간, 휴게소/맛집 정보를 한 줄 흐름으로 확인합니다.
- 상태 배지와 상세 정보를 동시에 보여줘서 판단 시간을 줄입니다.

### 2) 여정 인사이트
- 맞춤 팁은 날씨/교통 조건에 따라 메시지가 바뀝니다.
- 드라이브 플로우는 구간 버튼을 눌러 상세 안내를 확인합니다.

### 3) 가는 길 한눈에
- 네이버 Static Map으로 경로 미리보기 제공.
- 출발/도착 지점을 각각 마커로 표시.
- 교통 API 결과를 이용해 예상 소요시간/거리/도착 시간 표시.
- 경로 단계(드라이브 플로우)를 바로 확인할 수 있습니다.

### 4) 뉴스 & 현장 미디어
- 최신 뉴스/공지 링크를 큐레이션 목록으로 제공합니다.
- 네이버 이미지 검색 결과를 갤러리 탭으로 자동 로딩합니다.

### 5) 유튜브 하이라이트
- 유튜브 검색 결과를 카드로 2줄 슬라이더에 표시합니다.
- 좌/우 스크롤로 추가 영상 탐색이 가능합니다.

### 6) 여정 디테일
- 모임 정보, 일정, 식사, 공지사항을 한 섹션에 통합했습니다.

## 사용자 경험 설계 요약
- 스크롤에 따라 섹션이 전환되며 각 섹션이 화면을 가득 채웁니다.
- 정보 우선순위를 “실시간 → 경로/인사이트 → 미디어 → 디테일”로 구성.
- 불필요한 카드 남발을 줄이고 스트립/패널 중심 레이아웃으로 정리.
- 실질적으로 도움이 되는 정보(교통/도착 예상)에 집중.

## 외부 API 의존성
- OpenWeather: 날씨
- Naver Directions: 도로 소요시간
- Naver Static Map: 지도 미리보기
- Naver Image Search: 현장 사진 자동 갤러리
- YouTube Data API: 영상 추천
- Kakao Local API: 휴게소/맛집 (선택)

## 환경 변수
아래 키가 설정되어 있어야 자동 데이터가 표시됩니다.

- OPENWEATHER_KEY
- NAVER_MAPS_CLIENT_ID
- NAVER_MAPS_CLIENT_SECRET
- NAVER_SEARCH_CLIENT_ID
- NAVER_SEARCH_CLIENT_SECRET
- YOUTUBE_API_KEY
- KAKAO_REST_API_KEY (선택)
